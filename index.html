General Precision LGP-21 Emulator –
Hosting Site Home

This site hosts the current version of the retro-lgp21 emulator, an implementation of the General Precision LGP-21 computer system that runs in a web browser.

Main Links

  • LGP-21 Emulator Home Page
    The home page from which you can start the emulator.
  • Open Source Project
    Source code, documentation, and other resources for the retro-lgp21 emulator project at GitHub.
  • LGP-21 Documents and Software at bitsavers.org
    A collection of scanned copies of original LGP-21 manuals and other materials for the system. This project would not have been possible without the availability of these materials.
